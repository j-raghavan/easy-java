The for loop, dissected 🔁

Three parts: start ; keep-going test ; change. Press Step to watch it sum 1 through 5:

Sum.java
int total = 0;for (int i = 1; i <= 5; i++) { total = total + i;}System.out.println("Sum = " + total);

Read it aloud → "Start i at 1; while i ≤ 5, run the body; then add 1 to i." The loop ends when the test becomes false (i = 6).

Concept 03 · Interactive

Algorithm: find the maximum 🔍

Remember the biggest value seen so far, and update it when you find something larger. Press Step:

FindMax.java
// data = {3, 9, 2, 15} int max = data[0];for (int i = 1; i < data.length; i++) { if (data[i] > max) max = data[i];}System.out.println("Max = " + max);
i
—
max
—
// output
The pattern → Start max at the first element, then compare every other element to it. This "keep the best so far" idea also finds minimums, sums, and counts.
