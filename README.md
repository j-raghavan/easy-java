# Easy Java — Learn Java, the clear way

An interactive, self-narrating Java course. Every lesson explains itself and ships
with a live editor that really runs Java — so you don't just read, you change the
code, break it, and fix it.

## 🚀 Launch the course

### 👉 **[Open Easy Java](https://j-raghavan.github.io/easy-java/)**

> https://j-raghavan.github.io/easy-java/

Opens the Course Hub, where you can jump into any lesson.

## 💡 How it works

- **Self-narrating** — each lesson talks you through the material. Use the speaker
  button to mute and ↺ to replay any scene. Move with the **Next** button or arrow keys.
- **Live Java editor** — every lesson has an editor that really runs Java. Don't just
  read; change the code, break it, fix it. That's how it sticks.

## 🛠️ How this site is published

This is a static site served by **GitHub Pages** from the `master` branch. `index.html`
redirects to the Course Hub, and `.nojekyll` tells GitHub to serve the `.dc.html`
files as-is. To run it locally, serve the folder with any static server, e.g.:

```bash
python3 -m http.server 8000
# then open http://localhost:8000/
```
