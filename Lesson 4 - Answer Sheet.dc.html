Loops, Arrays & Methods: Answer Key

📌 For the student: Every task here mirrors an AP free-response pattern — loop over an array, inside a method, and return a result. Attempt them before reading.

1 Sum an array

Loop through {5, 10, 15, 20} and print the total.

int[] nums = {5, 10, 15, 20}; int total = 0; for (int i = 0; i < nums.length; i++) { total = total + nums[i]; } System.out.println("Total: " + total);
Expected outputTotal: 50
Key idea: the "accumulator" pattern — start total at 0, then add each element as you loop. The loop condition i < nums.length (not <=) keeps you inside the array's valid indexes 0–3.

Common mistake: using i <= nums.length — that tries to read nums[4], which doesn't exist, and crashes with ArrayIndexOutOfBoundsException.

2 Biggest of three

Write int biggest(int a, int b, int c) returning the largest.

int biggest(int a, int b, int c) { int max = a; if (b > max) { max = b; } if (c > max) { max = c; } return max; } System.out.println(biggest(4, 9, 7));
Expected output9
Key idea: same "best so far" pattern from the lesson — start max at a, then compare each other value and update. The method returns the result so the caller can use it.

Common mistake: forgetting the return, or declaring the method void. If it computes a value you want back, it must have a return type (int) and a return statement.

3 Count the even numbers

Method returns how many array values are even. Test on {2, 7, 4, 9, 10}.

int countEven(int[] nums) { int count = 0; for (int i = 0; i < nums.length; i++) { if (nums[i] % 2 == 0) { count = count + 1; } } return count; } int[] data = {2, 7, 4, 9, 10}; System.out.println("Even count: " + countEven(data));
Expected outputEven count: 3
Key idea: the "counter" pattern combines everything — a loop walks the array, an if tests each value with % 2 == 0, and a counter tallies the matches. 2, 4, and 10 are even → 3.
