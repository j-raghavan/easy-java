Concept 01

A program is a recipe 🍪

A computer follows instructions in order, top to bottom — never guessing, never skipping. If baking cookies were code, it'd look like this:

1Preheat oven to 350°
2Mix flour, sugar, butter
3Scoop dough onto tray
4Bake for 12 minutes
Why this matters → On the AP exam, "trace the code" questions ask you to be the computer: run each line in order and track what happens. That's literally the whole skill.

Practice · Assignment 1

Your assignment 📝

Do these in the live editor above (or in IntelliJ). Try each yourself first — then check the answer sheet to compare.

1
Warm-up. Create an int called score set to 90, and print "My score is 90" using the variable (not by typing 90 in the text).
2
Math. A movie ticket is $12.50 and you buy 3. Use a double for price and an int for quantity, then print the total cost.
3
Loop. Use a for loop to print the numbers 1 through 5, each on its own line.
Why do it yourself? → Reading code feels easy; writing it reveals what you actually understand. Every professional got good by making — and fixing — thousands of small mistakes.
